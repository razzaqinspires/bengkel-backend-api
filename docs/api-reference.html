Core API Reference

Dokumentasi teknis untuk integrasi Frontend (Mobile Apps & Web Dashboard).

GET

/workshops/nearby

Mencari bengkel dalam radius tertentu dari koordinat user.

Query Parameters

ParamTypeDescription
latFloatLatitude User (e.g. -6.200)
longFloatLongitude User (e.g. 106.816)
radiusIntJarak dalam KM (Default: 5)
