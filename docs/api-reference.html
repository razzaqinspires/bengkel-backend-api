CORE SYSTEM

API Endpoints

Kumpulan endpoint utama untuk mengelola Workshop (Bengkel), Booking Service, dan Pembayaran. Gunakan Token JWT untuk akses penuh.

Workshops Module

GET /workshops/nearby
PUBLIC ACCESS

Mencari bengkel terdekat berdasarkan koordinat GPS pengguna (Geospatial Query).

Query Parameters

ParamTypeRequiredDescription
latfloatYesLatitude User (e.g., -6.200)
longfloatYesLongitude User (e.g., 106.816)
radiusintNoJarak dalam KM (Default: 5)
 
curl "https://bengkel-backend-api.vercel.app/api/v1/workshops/nearby?lat=-6.1751&long=106.8650&radius=10"
POST /workshops
PARTNER ONLY

Mendaftarkan bengkel baru. Akun harus memiliki Role PARTNER.

Body Parameters

FieldTypeRequiredDescription
namestringYesNama Bengkel
addressstringYesAlamat Lengkap
latitudefloatYesKoordinat Lokasi
isEmergencyReadybooleanYesSiap panggilan darurat?
 
curl -X POST https://bengkel-backend-api.vercel.app/api/v1/workshops \
-H "Authorization: Bearer " \
-H "Content-Type: application/json" \
-d '{
  "name": "Bengkel Xarvionex Pusat",
  "address": "Jl. Merdeka No. 1, Jakarta",
  "latitude": -6.1750,
  "longitude": 106.8650,
  "isEmergencyReady": true
}'

Booking & Transaction

POST /bookings
AUTH REQUIRED

Membuat pesanan servis baru ke bengkel tertentu.

curl -X POST https://bengkel-backend-api.vercel.app/api/v1/bookings \
-H "Authorization: Bearer " \
-H "Content-Type: application/json" \
-d '{
  "workshopId": 12,
  "serviceIds": [1, 4],
  "note": "Ban bocor halus & Ganti Oli",
  "scheduledDate": "2026-03-01T10:00:00Z"
}'
