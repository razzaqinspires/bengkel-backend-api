Core API Reference

Dokumentasi endpoint utama untuk fitur bengkel dan transaksi.

GET /workshops/nearby

Mencari bengkel terdekat berdasarkan koordinat GPS.

Query Params

lat Latitude (float)
long Longitude (float)
radius Jarak KM (int)
curl "https://api-mexa.xarvionex.com/v1/workshops/nearby?lat=-6.2&long=106.8&radius=5"
POST /bookings

Membuat pesanan baru (Membutuhkan Header Authorization).

curl -X POST https://api-mexa.xarvionex.com/v1/bookings \
-H "Authorization: Bearer " \
-H "Content-Type: application/json" \
-d '{
  "workshopId": 12,
  "serviceIds": [1, 4],
  "note": "Ban bocor halus"
}'
