Authentication

Semua request ke endpoint private membutuhkan Header Authorization dengan format Bearer Token.

POST /auth/login 
curl -X POST https://api.bengkelin.com/api/v1/auth/login \
-H "Content-Type: application/json" \
-d '{
  "email": "mitra@bengkel.com",
  "password": "password123"
}'

Error 401

Unauthorized. Token tidak ada, kadaluarsa, atau tidak valid.

Error 403

Forbidden. Role akun tidak sesuai (User mengakses menu Mitra).
