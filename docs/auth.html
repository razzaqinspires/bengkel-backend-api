Autentikasi & Keamanan

Panduan mendapatkan akses token JWT untuk berinteraksi dengan API MEXA.

POST /auth/login

Gunakan endpoint ini untuk menukar kredensial menjadi Access Token.

curl -X POST https://api-mexa.xarvionex.com/v1/auth/login \
-H "Content-Type: application/json" \
-d '{
  "email": "user@example.com",
  "password": "password123"
}'
POST /auth/register

Mendaftarkan pengguna baru (User atau Partner).

curl -X POST https://api-mexa.xarvionex.com/v1/auth/register \
-H "Content-Type: application/json" \
-d '{
  "name": "Budi Santoso",
  "email": "budi@example.com",
  "password": "securePass123",
  "role": "USER",
  "phone": "08123456789"
}'
