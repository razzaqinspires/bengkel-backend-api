Autentikasi Sistem

Panduan lengkap mengelola akses pengguna dan mitra di dalam ekosistem MEXA.

Register Mitra Bengkel

POST

Endpoint untuk mendaftarkan akun baru sebagai pemilik bengkel (PARTNER).

curl -X POST https://mexa-api.xarvionex.com/api/v1/auth/register \
-H "Content-Type: application/json" \
-d '{
  "name": "Arifi Owner",
  "email": "arifi@mexa.com",
  "password": "securePass123!",
  "role": "PARTNER",
  "phone": "081234567890"
}'

Login Access

POST

Gunakan endpoint ini untuk mendapatkan accessToken (Berlaku 15 menit) dan refreshToken.

curl -X POST https://mexa-api.xarvionex.com/api/v1/auth/login \
-H "Content-Type: application/json" \
-d '{
  "email": "arifi@mexa.com",
  "password": "securePass123!"
}'
Penting: Simpan Token di Secure Storage (Mobile) atau HttpOnly Cookie (Web). Jangan simpan di LocalStorage biasa.
