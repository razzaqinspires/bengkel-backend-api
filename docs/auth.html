SECURE LAYER

Autentikasi Sistem

Setiap request ke endpoint privat membutuhkan Bearer Token (JWT) pada Header Authorization. Token ini didapatkan setelah login berhasil.

Security Best Practice

Jangan pernah menyimpan Access Token di LocalStorage browser jika memungkinkan. Gunakan HttpOnly Cookie atau Secure Storage pada perangkat mobile untuk mencegah serangan XSS.

POST /auth/login
PUBLIC ACCESS

Menukarkan kredensial (email & password) untuk mendapatkan Access Token & Refresh Token.

Body Parameters

FieldTypeRequiredDescription
emailstringYesEmail terdaftar
passwordstringYesPassword user
 
curl -X POST https://bengkel-backend-api.vercel.app/api/v1/auth/login \
-H "Content-Type: application/json" \
-d '{
  "email": "user@example.com",
  "password": "password123"
}'
POST /auth/register
PUBLIC ACCESS

Mendaftarkan pengguna baru sebagai User (Pemilik Kendaraan) atau Partner (Mitra Bengkel).

Body Parameters

FieldTypeRequiredDescription
namestringYesNama Lengkap
emailstringYesEmail Unik
passwordstringYesMin 8 karakter
rolestringYes'USER' | 'PARTNER'
phonestringYesNomor HP Aktif
 
curl -X POST https://bengkel-backend-api.vercel.app/api/v1/auth/register \
-H "Content-Type: application/json" \
-d '{
  "name": "Arifi Razzaq",
  "email": "arifi@example.com",
  "password": "SecurePassword123!",
  "role": "PARTNER",
  "phone": "081299998888"
}'
